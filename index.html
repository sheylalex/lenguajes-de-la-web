Algunos Lenguajes de la Web

HTML + CSS + Javascript

  • HTML.5

    • Es un lenguaje es se utiliza comúnmente para establecer la estructura y contenido de un sitio web, tanto de texto, objetos e imágenes.

  • CSS . 3

    • Podemos decir que el lenguaje CSS sirve para organizar la presentación y aspecto de una página web.

  • Javascript

    • JavaScript (abreviado comúnmente JS) es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.
